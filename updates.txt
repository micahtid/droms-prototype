# Latest Updates Summary

## Map View Enhancements
- Filter dialog now supports multi-select for disaster types and severity levels
- Filter dialog automatically closes when clicking anywhere on the map
- Filter dialog width increased for better usability
- Added severity filter alongside disaster type filter
- Legend moved to bottom-left and made collapsible (collapsed by default)
- Legend displays disaster types in Notion-style tags with glassmorphism effect
- Removed FERPA toggle from legend

## Filter & Tag Styling
- Updated all filter tags to Notion-style design with colored backgrounds
- Selected tags show disaster type/severity colors with 80% opacity backgrounds
- Tag text uses white color with 70% opacity for better readability
- Removed colored circle indicators, replaced with full tag highlighting
- Clear Filters button styled with white background (previously red)

## List View Improvements
- Added search bar at the top for filtering disasters by type, location, or description
- Added filter menu matching Map View functionality (types and severity)
- Filter menu closes when clicking outside
- Filters shared between Map View and List View for consistency

## Polygon Updates
- Updated disaster zone polygons to rectangular shapes with 90-degree angles only
- Adjusted polygon coordinates to properly encompass all related disaster pin points
- Tornado polygon: L-shaped rectangle covering all 4 sub-disasters
- Flood polygon: L-shaped rectangle covering all 3 sub-disasters

## Issue Details Reorganization
- Removed "Reviewed" status badges throughout the application
- Reorganized content layout for better information flow:

  - Parent disaster context badge with glassmorphism style
  - Key info cards (Severity & Last Updated) in 2-column grid
  - Removed icons from Severity and Last Updated cards
- Added city and state to location display (e.g., "Lincoln, NE")
- Updated "Part of" tag with glass/frosted style, visible blue text, and subtle border

## Deployment Detail Updates
